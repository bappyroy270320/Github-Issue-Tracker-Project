1️⃣ What is the difference between var, let, and const?
Ans:
1.Var--
A variable declared with var can be re-declared and updated. and it is function scoped
2.let--
It can be updated but cannot be declared again in the same scope and it is block scoped
3.const--
It cannot be updated or redeclared after initialization. and it also block scoped.

2️⃣ What is the spread operator (...)?
Ans:
The spread operator (...) is used to expand or copy elements from an array or object.

3️⃣ Difference between map(), filter(), and forEach()
Ans:
1.map()--
Creates a new array and Transforms each element of the array.
2.filter()--
Creates a new array and Returns elements that match a condition.
3.forEach()--
Loops through each element and Does not return a new array.

4️⃣ What is an arrow function?
Ans:
An arrow function is a shorter way to write functions in JavaScript.
It uses the => symbol and makes the code cleaner and easier to read.

5️⃣ What are template literals?
Ans:
Template literals allow us to create strings easily and insert variables inside strings.
They use backticks ( ) instead of quotes.
